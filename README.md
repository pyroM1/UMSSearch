# UMSSearch
Search plugin for Universal Media Server

# Usage
Folders repesentation

`..`       | [Search...]
-----------|------------
`[abc]`    | `[def]`
`[ghi]`	   | `[jkl]`
`[mno]`    | `[pqr]`
`[stuv]`   | `[wxyz]`
`[. _-]`   | `[0-9]`

to search "my movie.mkv"

[ **m** no] > [wx **y** z] > [.  _-] > [ **m** no] > [mn **o** ] > [stu **v** ] > [Search...]

Result

**my mov** ie.mkv
